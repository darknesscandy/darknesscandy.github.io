    基于M5STACK的网络时钟

    发表于 2021-03-07

    某天闲来无事，突觉桌面需要一个时钟以便于随时查看时间。手头51单片机没学通，淘宝上的套件价格又有些贵，一通翻找后沉睡许久的M5STACK Core映入眼帘，或许它能满足我的需求。一番百度大法了解之后，便有了今天这篇文章。

    一、项目功能

    1. 所用技术

    • 本项目全在电脑端UIFlow中完成，只需理清基本逻辑即可。

    2. 实现功能

    • 基本的时钟，日历功能。
    • 由于M5STACK可拓展性，可以增添温湿度显示等功能。

    二、所需材料

    • M5STACK Core 一块
    • Type C 数据线一根
    • 电脑一台

    三、环境搭建

    首先需要下载M5Burner烧录软件和CP2104 Driver驱动（即图中软件栏第二和第三个），地址https://m5stack-store.myshopify.com/pages/download，根据系统不同选择对应版本，这里win10为例。
